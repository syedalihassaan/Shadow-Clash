# Shadow Clash ⚔️

Shadow Clash is a fast-paced 2-player fighting game built with HTML, CSS, and JavaScript. Battle as Ryu or Ken in a dark neon arena and defeat your opponent using punches, kicks, blocks, and powerful combos.

## 🎮 Features

- Local 2-player gameplay
- Punch, kick, jump, and block mechanics
- Combo attack system
- Health bars and round timer
- Best-of-three match format
- Dynamic visual effects and animations
- Responsive keyboard controls

## 🕹️ Controls

### Player 1 (Ryu)
| Action | Key |
|----------|-----|
| Move Left | A |
| Move Right | D |
| Jump | W |
| Punch | F |
| Kick | G |
| Block | S |

### Player 2 (Ken)
| Action | Key |
|----------|-----|
| Move Left | ← |
| Move Right | → |
| Jump | ↑ |
| Punch | K |
| Kick | L |
| Block | ↓ |

## 🚀 How to Play

1. Open `ShadowClash.html` in your browser.
2. Click **Start Fight**.
3. Use the controls to attack, defend, and perform combos.
4. Win 2 rounds to become the champion.

## 🏆 Objective

Reduce your opponent's health to zero before the timer runs out. The first fighter to win two rounds wins the match.
